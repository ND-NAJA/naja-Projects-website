Hairline-divided list of feature bullets, each prefixed with a mono `→` arrow — or, when given a `detail`, a `+`/`−` marker that expands to reveal extra copy on click.

```jsx
<FeatureList onDark={false} items={[
  "Document management, shared across every job and client",
  { text: "Planning that connects sales, delivery and site work", detail: "Sales, scheduling and site notes all live on the same job record." },
]} />
```
