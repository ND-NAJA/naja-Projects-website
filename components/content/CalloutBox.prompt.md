Thin-bordered aside box used next to detail sections ("How a session works", "Why timber frame", "Before you write").

```jsx
<CalloutBox title="How a session works" onDark>
  <p>No slide decks. You bring the decision you're stuck on...</p>
</CalloutBox>
```

Set `onDark` when placed on a navy section; omit on paper (white) sections — border/background opacity and text color flip accordingly.
